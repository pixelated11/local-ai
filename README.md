# Local Assistant
Your own, locally-run, everyday-task local assistant. Interface made with python, and using the Qwen3 4B (4-bit quantized version) LLM.

## Platform support
Note that Local Assistant only supports **Linux** at this time, and not Windows. Support for windows will be added soon, in another branch in this repository.

You can find all those, in the documentation I made [here](https://pixelated11.github.io/local-assistant-docs)
